## コードへの組み込み方（バトルシーン）

スプライト作成後に `scenes/battle_scene.py` の以下2箇所を `rectb` から `blt` に変更する：

```python
# 敵モンスター（現在：rectb のみ）
# u = ((spec_id - 1) % 4) * 64
# v = 16 + ((spec_id - 1) // 4) * 64
pyxel.blt(8, _ENEMY_PANE_Y + 8, 1, u, v, 64, 64, 0)

# 味方モンスター
pyxel.blt(160, _PLAYER_PANE_Y + 8, 1, u, v, 64, 64, 0)
```

## 制作の進め方

1. **最優先**：モンスタースプライト 15体（バンク1、各64×64）を作成
2. `create_assets.py` の `main()` 内に `_draw_monster(img1, spec_id, u, v)` を追加
3. `uv run python create_assets.py` で再生成
4. `scenes/battle_scene.py` の `rectb` → `blt` に切り替え
5. タイルセット・マップを徐々に改善

> Pyxelエディタ（`uv run pyxel edit assets/game.pyxres`）で直接描くのが最も直感的。
